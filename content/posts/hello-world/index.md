## Code Highlighting

Hugo has built-in syntax highlighting via Chroma:

```python
def hello():
    print("Hello from my new blog!")

if __name__ == "__main__":
    hello()
```

```javascript
const greet = (name) => {
  console.log(`Hello, ${name}!`);
};

greet("world");
```
